# Sentinel AI

Autonomous Vision Agent — система компьютерного зрения реального времени.

## Что умеет
- Захват видео с камеры
- Детекция объектов через YOLOv8
- Трекинг объектов с уникальными ID
- Счётчик людей в кадре в реальном времени

## Стек
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- ByteTrack

## Запуск
```
python sentinel.py
```
